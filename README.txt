HOW TO RUN ON ANY WINDOWS MACHINE
══════════════════════════════════

STEP 1 — Install Python (if not already installed)
  → Download from: https://www.python.org/downloads/
  → Choose Python 3.10 or newer
  → During installation, CHECK the box: "Add Python to PATH"

STEP 2 — Install Node.js (if not already installed)
  → Download from: https://nodejs.org/
  → Choose the LTS (Long Term Support) version

STEP 3 — Run the Setup (only once!)
  → Double-click:  setup.bat
  → This installs all required packages automatically
  → Wait for it to say "Setup Complete!"

STEP 4 — Launch the App
  → Double-click:  run_app.bat
  → Two terminal windows will open (backend + frontend)
  → Your browser will open automatically at http://localhost:5173

STOPPING THE APP
════════════════
  → Close the two terminal windows that opened
    (titled "AI Formater — Backend" and "AI Formater — Frontend")

SYSTEM REQUIREMENTS
═══════════════════
  Operating System : Windows 10 or Windows 11
  Python           : 3.10 or newer
  Node.js          : 18 or newer
  Internet          : Required during setup only (for downloading packages)
  Disk Space       : ~500 MB (for all packages)

TROUBLESHOOTING
═══════════════
  ● "python not found" → Re-install Python and check "Add to PATH"
  ● "node not found"   → Re-install Node.js
  ● Port already in use → Restart your computer and try again
  ● Setup failed        → Delete the file ".setup_done" and run setup.bat again
